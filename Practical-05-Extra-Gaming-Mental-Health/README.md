Extra Practical 05

Aim:
To implement the Random Forest algorithm on the Gaming and Mental Health dataset and evaluate the model performance using appropriate metrics.

Description:
This extra practical explores the use of the Random Forest algorithm on a real-world dataset related to gaming habits and mental health indicators. The objective is to build a predictive model capable of identifying patterns and relationships between gaming behavior and mental health outcomes.

The workflow includes:
• Loading and inspecting the dataset
• Data cleaning and preprocessing
• Training a Random Forest classification model
• Generating predictions for the target variable
• Evaluating the model using accuracy score, confusion matrix and classification metrics

This experiment demonstrates how ensemble machine learning techniques can improve model stability and predictive capability when working with complex behavioral datasets.
